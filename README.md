# 🧮 命令行计算器

一个用 Python 写的简易计算器，支持加、减、乘、除四种运算，交互式命令行操作。

## 🚀 运行方式

```bash
python calculator.py
```

## 📖 使用说明

```
>>> 5 + 3
5.0 + 3.0 = 8

>>> 10 - 4
10.0 - 4.0 = 6

>>> 6 * 7
6.0 * 7.0 = 42

>>> 15 / 3
15.0 / 3.0 = 5.0

>>> quit
再见！👋
```

## 🛠️ 用到的知识点

- 函数定义与调用
- 字符串拆分
- 条件判断（if / elif / else）
- 异常处理（try / except）
- 类型转换
- while 循环
