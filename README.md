一个用 Python 写的简易计算器，支持加、减、乘、除四种运算，交互式命令行操作。

## ✨ 功能

- ➕ 加法
- ➖ 减法
- ✖️ 乘法
- ➗ 除法
- 🔄 异常处理（除零、非法输入）

## 🚀 快速开始

### 依赖
```bash
无额外依赖（仅用标准库）
```

### 运行
```bash
python calculator.py
```

## 📖 使用说明

运行后输入算式（如 `5 + 3`），回车查看结果。输入 `quit` 退出。

## 🛠️ 用到的知识点

| 知识点 | 应用 |
|--------|------|
| 函数定义与调用 | 封装运算逻辑 |
| 字符串拆分 | 解析输入算式 |
| 条件判断 | 识别运算符 |
| 异常处理 | 处理除零和非法输入 |
| while循环 | 持续交互 |
